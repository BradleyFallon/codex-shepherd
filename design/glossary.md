# Glossary

Planner:
The persistent reasoning agent responsible for task selection and decomposition.

Codex:
The execution agent responsible for modifying code and running tests.

Shepherd:
The deterministic control process that orchestrates planner and executor.

Objective:
A verifiable condition derived from documents that must be satisfied.

Task:
A concrete action intended to satisfy an objective.

Blocked:
A state indicating execution cannot proceed without human input.

Authoritative Document:
A document whose requirements must be implemented exactly.
